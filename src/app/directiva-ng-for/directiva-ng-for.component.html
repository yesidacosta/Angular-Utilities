Directiva ngFor

DIRECTIVA NgForOf o ngFor

Una directiva estructural representa un ciclo para recorrer cada elemento de una colección. La directiva se coloca en un elemento y este se convierte en el padre de los elementos renderizados de la colección.

Para usar la directiva ngFor {{ "*ngFor = let item of items; index as i; trackBy: trackByFn" }}

  • items: Colección
  • item: Elemento de la colección
  • index: Indice generado por el ciclo
  • i: Indice asignado desde index
  • trackBy: Clave de función de seguimiento
  • trackByFn: Función de seguimiento
