INTERPOLACIÒN DE VARIABLES

Caracteristica para embeber expresiones que pueden ser.

  • Strings
  • Operaciones matemàticas bàsicas
  • Mètodos que devuelven valores
  • Expresiones lògicas

Las expresiones se encierran entre {{ "{{ }}" }}

La interpolaciòn se hace desde el controlador hacia la vista, No deben contener sentencias que afecten el componente como:

  • Asignación
  • Incrementos
  • Decrementos

Si no se cumple, devuelve un error

Ejemplo

Interpolar variable
Código Resultado
{{"\

{{" + "variableInterpolada" + "}}\

"}}		 {{variableInterpolada}}
{{"\

{{" + "1 + 7" + "}}\

"}}		 {{1 + 7}}
{{"\

{{" + "true" + "}}\

"}}		 {{ true }}
{{ getCadena }} {{ getCadena() }}
