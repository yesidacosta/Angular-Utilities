Directiva ngIF

DIRECTIVA ngIF

Una directiva estructural que incluye condicionalmente una plantilla basada en el valor de una expresión forzada a booleana

Para usar la directiva ngIf {{ "*ngIf = 'Condición'" }}

Para usar la directiva ngIf en un ng-template {{ "[ngIf] = 'Condición'" }}
